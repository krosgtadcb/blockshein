CAMBIO DE DIFICULTAD (99%)

  IMPACTO EN MINERÍA:

  Difficulty 4:
  ┌─────────────────────────────────────────┐
  │ Busca hash que comience con: 0000       │
  │ Promedio intentos: 4^16 = 4 mil         │
  │ Tiempo: 10-30 segundos en CPU moderno   │
  └─────────────────────────────────────────┘

  Difficulty 99:
  ┌─────────────────────────────────────────┐
  │ Busca hash que comience con: 99 ceros   │
  │ Promedio intentos: 16^99 = EXTREMO      │
  │ Tiempo: 5-30 MINUTOS en CPU moderno     │
  │ Nota: Puede ser más con CPU débil       │
  └─────────────────────────────────────────┘

  FÓRMULA:
  Intentos = 16^(dificultad)
  
  Ejemplo:
  difficulty=4:  16^4 = 65,536 intentos
  difficulty=10: 16^10 = 1,099,511,627,776 intentos
  difficulty=99: 16^99 = ???? (números astronómicos)

  NOTA: 99% de dificultad es EXTREMADAMENTE desafiante.
        Si quieres pruebas rápidas, usa difficulty=10-15.
